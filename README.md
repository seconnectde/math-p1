# Math P1 Practice Prototype

โปรเจกต์นี้เป็น prototype สำหรับทดลองไอเดียเว็บแอปฝึกคณิตศาสตร์สำหรับเด็กชั้น ป.1 โดยเน้นการสำรวจรูปแบบการเรียนรู้ผ่านแบบฝึกสั้น ๆ ที่เด็กสามารถทำเองได้ และผู้ปกครองหรือครูสามารถดูผลลัพธ์เบื้องต้นได้

## เป้าหมายของโปรเจกต์

- ทดลองแนวคิดแอปฝึกคณิตศาสตร์สำหรับเด็กเล็ก
- สำรวจ flow การทำแบบฝึก ตอบคำถาม ตรวจคำตอบ และดูคะแนน
- ทดลองระบบบันทึกความคืบหน้าในเครื่องผู้ใช้
- ทดลองการ Export ผลลัพธ์เป็น PDF สำหรับเก็บหรือส่งต่อ
- ใช้เป็นพื้นที่ลองไอเดียด้าน UX, เนื้อหาแบบฝึก และการติดตามพัฒนาการของเด็ก

## สถานะโปรเจกต์

โปรเจกต์นี้ยังเป็นเวอร์ชันทดลอง ไม่ใช่ผลิตภัณฑ์สมบูรณ์ ฟีเจอร์ เนื้อหาแบบฝึก และหน้าตาแอปยังสามารถเปลี่ยนแปลงได้ตามการทดลองและ feedback

ตอนนี้เป็น static web app ที่เขียนด้วย HTML, CSS และ JavaScript ล้วน ๆ ยังไม่มี backend, login, ระบบจัดการผู้ใช้ หรือฐานข้อมูลกลาง

## สิ่งที่มีใน prototype

- แบบฝึกคณิตศาสตร์ ป.1 จำนวน 10 ชุด
- แต่ละชุดมี 50 ข้อ รวมทั้งหมด 500 ข้อ
- หัวข้อครอบคลุมบวก ลบ บวก-ลบผสม เปรียบเทียบจำนวน เรียงลำดับจำนวน หลักสิบหลักหน่วย รูปทรง เงิน เวลา และโจทย์รวมทบทวน
- ตรวจคำตอบและแสดงคะแนนรายชุด
- แสดงสรุปข้อถูก ข้อผิด และเฉลยข้อที่ตอบผิด
- หน้าแรกแสดงจำนวนข้อที่ทำแล้ว จำนวนข้อที่ตอบถูก และชุดที่ทำได้ดีที่สุด
- บันทึก progress ด้วย `localStorage` และ temp save อัตโนมัติทุก 30 วินาที
- Export รายงานผลผ่านระบบ print/PDF ของเบราว์เซอร์
- สุ่มโจทย์ชุดใหม่ได้หลังทำครบ ตรวจครบ และ Export ผลแล้ว
- รองรับการใช้งานบนมือถือเบื้องต้นด้วยเมนูและแถบควบคุมแบบ responsive

## แนวทางที่อยากทดลองต่อ

- ปรับระดับความยากตามผลการทำแบบฝึก
- เพิ่มเสียง ภาพ หรือ animation เพื่อช่วยให้เด็กสนุกขึ้น
- เพิ่มโหมดสำหรับผู้ปกครองหรือครู
- ออกแบบระบบติดตามพัฒนาการระยะยาว
- แยกชุดแบบฝึกตามทักษะหรือเป้าหมายการเรียนรู้ที่ละเอียดขึ้น
- ทดสอบกับเด็กจริงเพื่อดูว่า flow เข้าใจง่ายพอไหม

## วิธีเปิดใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้ทันที หรือรัน local server:

```bash
python3 -m http.server 8000
```

แล้วเข้า:

```text
http://localhost:8000
```

## โครงสร้างไฟล์

```text
.
├── index.html   # โครงหน้าเว็บและ template ของหน้าแรก/หน้าแบบฝึก
├── styles.css   # หน้าตา responsive layout และ print style
├── app.js       # logic สร้างโจทย์ ตรวจคำตอบ บันทึก progress และ export
└── README.md    # เอกสารโปรเจกต์
```

## หมายเหตุ

ข้อมูลทั้งหมดถูกเก็บไว้ในเบราว์เซอร์ของผู้ใช้ผ่าน `localStorage` ไม่ได้ส่งข้อมูลไป server ภายนอก หากล้างข้อมูลเบราว์เซอร์หรือเปลี่ยนเครื่อง ความคืบหน้าจะไม่ตามไปด้วย
