# สอนคณิตศาสตร์ ป.1

เว็บแบบฝึกคณิตศาสตร์สำหรับเด็กชั้น ป.1 ใช้งานได้จากไฟล์ static ล้วน ๆ เหมาะสำหรับเปิดในเบราว์เซอร์ทันทีโดยไม่ต้องติดตั้ง dependency หรือรัน build step

## ฟีเจอร์ล่าสุด

- มีแบบฝึกทั้งหมด 10 ชุด ชุดละ 50 ข้อ รวม 500 ข้อ
- ครอบคลุมการบวก ลบ บวก-ลบผสม เปรียบเทียบจำนวน เรียงลำดับจำนวน หลักสิบหลักหน่วย รูปทรง เงิน เวลา และโจทย์รวมทบทวน
- หน้าแรกสรุปจำนวนข้อที่ทำแล้ว จำนวนข้อที่ตอบถูก และชุดที่ทำได้ดีที่สุด
- แต่ละชุดมีคะแนน แถบความคืบหน้า สรุปข้อถูกผิด และรายการข้อที่ตอบผิดพร้อมเฉลย
- บันทึกความคืบหน้าอัตโนมัติด้วย `localStorage` พร้อม temp save ทุก 30 วินาที และข้อมูล temp หมดอายุใน 2 วัน
- กดบันทึกเองได้ ตรวจคำตอบได้ และเริ่มชุดเดิมใหม่ได้
- Export PDF ผ่านหน้าต่าง print ของเบราว์เซอร์ พร้อมรายงานคะแนน คำตอบที่ทำ คำตอบที่ถูก และผลรายข้อ
- สุ่มโจทย์ชุดใหม่ได้หลังทำครบ 50 ข้อ ตรวจครบ และยืนยันว่า Export PDF แล้ว
- รองรับมือถือด้วยเมนูชุดแบบฝึกแบบพับได้ และแถบปุ่มควบคุมแบบ fixed bottom

## วิธีใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ แล้วเลือกชุดแบบฝึกจากหน้าแรกหรือเมนูด้านบน

ลำดับการทำงานที่แนะนำ:

1. เลือกชุดแบบฝึก
2. ตอบคำถามให้ครบ 50 ข้อ
3. กด `ตรวจคำตอบ`
4. ดูคะแนนและทบทวนข้อที่ผิด
5. กด `Export PDF` เพื่อบันทึกหรือพิมพ์รายงาน
6. หลัง Export แล้วจึงกด `สุ่มโจทย์ใหม่` ได้

## ชุดแบบฝึก

| ชุด | หัวข้อ | เนื้อหา |
| --- | --- | --- |
| 1 | บวกไม่เกิน 10 | รวมจำนวนเล็ก ๆ |
| 2 | ลบไม่เกิน 10 | ลบแบบคำตอบไม่ติดลบ |
| 3 | บวกและลบผสม | ฝึกอ่านเครื่องหมายและเลือกวิธีคิด |
| 4 | มากกว่า น้อยกว่า เท่ากับ | เลือก `>`, `<` หรือ `=` |
| 5 | เรียงลำดับจำนวน | เติมจำนวนที่หายไป |
| 6 | บวกไม่เกิน 20 | รวมจำนวนที่ใหญ่ขึ้น |
| 7 | ลบไม่เกิน 20 | ลบจากจำนวน 10-20 |
| 8 | หลักสิบ หลักหน่วย | แยกค่าประจำหลักของจำนวนสองหลัก |
| 9 | รูปทรง เงิน และเวลา | โจทย์ชีวิตประจำวัน |
| 10 | โจทย์รวมทบทวน | รวมหลายทักษะในชุดเดียว |

## โครงสร้างไฟล์

```text
.
├── index.html   # โครงหน้าเว็บและ template ของหน้าแรก/หน้าแบบฝึก
├── styles.css   # หน้าตา responsive layout และ print style
├── app.js       # logic สร้างโจทย์ ตรวจคำตอบ บันทึก progress และ export
└── README.md    # เอกสารโปรเจกต์
```

## การพัฒนา

โปรเจกต์นี้เป็น vanilla HTML, CSS และ JavaScript จึงไม่ต้องติดตั้ง package เพิ่ม

ถ้าต้องการเปิดผ่าน local server เพื่อทดสอบ:

```bash
python3 -m http.server 8000
```

แล้วเข้า `http://localhost:8000`

## หมายเหตุด้านข้อมูล

ความคืบหน้าจะถูกเก็บไว้ในเบราว์เซอร์เครื่องนั้น ๆ ผ่าน `localStorage` ไม่ได้ส่งข้อมูลไป server ภายนอก หากล้างข้อมูลเบราว์เซอร์หรือเปลี่ยนเครื่อง ความคืบหน้าจะไม่ตามไปด้วย
